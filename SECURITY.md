# Security Policy

## Supported Versions

The following versions of `airports-py` are currently receiving security updates:

| Version | Supported |
| ------- | --------- |
| 3.1.x   | Yes       |
| 3.0.x   | Yes       |
| 2.x.x   | No        |
| 1.x.x   | No        |

If you are using an unsupported version, please upgrade to the latest release.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by emailing **aashishvanand@gmail.com**. Do not open a public issue.

When reporting, please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue, with code samples if possible
- The affected version(s) of the library

## Response Process

1. **Acknowledgement** - You will receive a response within 3-5 business days confirming receipt.
2. **Assessment** - The vulnerability will be validated, reproduced, and assessed for severity.
3. **Fix** - A patch will be developed and released. You may be contacted for further details.
4. **Disclosure** - A security advisory will be published on GitHub after the fix is released.

Thank you for helping keep this project and its users safe.
